Hexo备忘录

2019/09/14

  • 生成密钥并添加

    ssh-keygen -t rsa -C "email"
# 拉取到本地
git clone git@github.com:JacksonLeon/Hexo_Blog.git Blog

  • 安装Hexo

    # 安装nvm版本管理（直接安装nodejs会有权限错误）
yay -S nvm
source /usr/share/nvm/init-nvm.sh
nvm install node
    
# 安装npm和hexo
sudo pacman -S npm
npm install hexo-cli -g
sudo npm install hexo -g
# 安装依赖
sudo npm install

  • Git

    # git默认设置
git config --global user.email "im.lauzh@outlook.com"
git config --global user.name "lau"
# 切换分支deploy
git checkout -b deploy
# 修改及添加remote
git remote rename origin github
git remote add gitee git@gitee.com:JacksonLeon/Hexo_Blog.git
git remote add gitlab git@gitlab.com:JosephLau/hexo_blog.git

  • 其他
